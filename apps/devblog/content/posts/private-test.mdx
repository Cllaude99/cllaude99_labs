# Private 글 테스트

이 글은 비밀번호로 보호되어 있습니다. 이 내용을 볼 수 있다면 인증에 성공한 것입니다.

## 테스트 항목

- PIN 입력 모달이 정상적으로 표시되는지
- 잘못된 비밀번호 입력 시 shake 애니메이션이 동작하는지
- 올바른 비밀번호(1234) 입력 시 컨텐츠가 표시되는지
- sessionStorage에 인증 상태가 저장되는지
- 같은 탭에서 다른 페이지 갔다가 돌아왔을 때 바로 컨텐츠가 표시되는지
